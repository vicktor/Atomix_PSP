Controls
------------------------------------------------------------------------------

In main menu:

Cross: Select Option

In game/editor:

Cross: Select/deselect atom or put tile
Start: Exit game and return to main menu
Up Down Left Right to move atom or move pointer 

Select: In editor save level

In editor:

Cross: Select/deselect atom or put tile
Triangle: delete tile
Start: Exit Editor and return to main menu
Up Down Left Right: to move atom or move pointer 
Select: Save level
Left/Right Trigger: Change level


You can change all backgrounds also, simple copy new background with name bg0 to bg3
